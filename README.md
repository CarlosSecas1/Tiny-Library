# Library System

A simple command-line library system built with Python using object-oriented programming.

## Features
- View available and unavailable books
- Borrow a book
- Return a book
- View borrowed titles
- Track active loans

## Classes
- **Book**: stores the title and checked-out status of a book
- **Member**: stores the member name and borrowed book titles
- **Loan**: tracks the loan status between a member and a book

## How to Run
1. Make sure Python is installed
2. Run the program:

```bash
python main.py
